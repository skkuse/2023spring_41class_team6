C++는 다양한 데이터형을 제공하여 변수를 선언하고 값을 할당할 수 있습니다. 이번에는 C++에서 제공하는 주요 데이터형에 대해 살펴보겠습니다.

1. 정수형

정수형은 -2,147,483,648부터 2,147,483,647까지의 값을 저장할 수 있습니다. 예를 들어 변수 x를 정수형으로 선언하려면 다음과 같이 하면 됩니다.

```
int x;
```

2. 실수형

실수형은 소숫점 이하의 값을 가지는 수를 저장할 수 있습니다. float과 double이라는 두 가지 유형의 실수형이 있습니다. 예를 들어 변수 y를 double 실수형으로 선언하려면 다음과 같이 하면 됩니다.

```
double y;
```

3. 문자형

문자형은 단일 문자를 저장하는 데 사용합니다. char는 하나의 문자만 저장할 수 있습니다. 예를 들어 변수 z를 문자형으로 선언하려면 다음과 같이 하면 됩니다.

```
char z;
```

4. 논리형

논리형은 변수가 참 또는 거짓인지를 나타내기 위해 사용됩니다. 다음과 같이 변수 a를 논리형으로 선언할 수 있습니다.

```
bool a;
```

위와 같이 C++에서 제공하는 데이터형을 이용하여 변수를 선언할 수 있습니다. 변수의 값은 초기화나 입력 및 연산을 통해 언제든지 변경할 수 있습니다. 이에 대한 내용은 이후에 다루어질 예정입니다.
