JavaScript는 웹 프로그래밍에서 가장 널리 사용되는 언어입니다. 변수는 JavaScript에서 매우 중요한 개념 중 하나이며, 데이타를 저장하고 다루는 데 사용됩니다. 이번에는 JavaScript에서 변수에 대해 자세히 알아보도록 하겠습니다.

JavaScript에서 변수를 정의하는 방법은 다음과 같습니다. 'var' 키워드를 변수 이름 앞에 붙여서 변수를 선언합니다. 다음은 변수를 선언하는 방법의 예입니다.

```javascript
var quantity;
```

이제 'quantity'라는 이름의 변수가 선언되었습니다. 변수가 선언된 다음에는 값을 지정하여 초기화할 수 있습니다. 다음은 값을 지정하여 변수를 초기화하는 방법입니다.

```javascript
var quantity = 10;
```

모든 JavaScript 변수는 이름에 따라 데이터 형식을 자동으로 타입처럼 사용합니다. 이를 '동적 데이터 형식(dynamically typed)'이라고 합니다. 즉, JavaScript 엔진은 변수에 담겨 있는 값의 유형을 자동으로 감지하고 이에 맞게 처리합니다. 따라서 정수, 부동소수점, 문자열 등 모든 유형의 데이터를 변수에 저장할 수 있습니다.

JavaScript에서 변수는 다음과 같이 할당도 가능합니다.

```javascript
var quantity1 = 10;
var quantity2 = quantity1;
```

위의 코드는 'quantity1' 변수에 '10'을 할당한 뒤, 'quantity2' 변수에 'quantity1' 변수를 할당한 것입니다. 이제 'quantity2'에는 'quantity1'에 할당된 값 '10'이 저장됩니다.

이번에는 변수의 값을 변경하는 방법을 살펴보겠습니다. 변수는 새 값을 할당함으로써 값을 변경할 수 있습니다. 다음은 값을 변경하는 코드 예제입니다.

```javascript
var quantity = 10;
quantity = 20;
```

이렇게 하면 변수 'quantity'의 값이 '10'에서 '20'으로 변경됩니다.

JavaScript에서는 변수 이름에 사용할 수 있는 문자 또는 숫자의 제한이 없습니다. 심지어는 달러 기호 또는 밑줄 기호도 변수 이름에 사용할 수 있습니다. 하지만 변수 이름은 대소문자를 구분합니다. 따라서 'quantity'와 'Quantity'는 서로 다른 변수입니다.

이상으로 변수 선언, 초기화, 할당 등에 대한 JavaScript에서의 기본적인 개념과 사용 방법을 알아 보았습니다. 이를 바탕으로 JavaScript에서 다양한 작업을 수행할 수 있습니다.
