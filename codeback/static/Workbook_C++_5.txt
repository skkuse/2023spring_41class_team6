안녕하세요. C++ 언어의 연산자에 대해 설명해드리겠습니다.

연산자는 C++에서 가장 기본이 되는 것 중 하나입니다. 연산자는 더하기, 빼기, 나누기, 곱하기 등의 동작을 수행합니다. 변수나 상수 등의 값을 조작하거나 할당하는 등의 기능 또한 가지고 있습니다.

+ : 더하기 연산자
- : 빼기 연산자
* : 곱하기 연산자
/ : 나누기 연산자
% : 나머지 연산자

위 연산자들은 산술 연산을 수행합니다.

그리고 비교 연산자, 대입 연산자 같이 연산의 결과가 참, 거짓 등의 논리 값을 반환하는 연산자도 있습니다.

== : 좌항과 우항이 같은지 확인하는 동등 비교 연산자
!= : 좌항과 우항이 다른지 확인하는 부등 비교 연산자
> : 좌항이 우항보다 큰지 확인하는 대소 비교 연산자
< : 좌항이 우항보다 작은지 확인하는 대소 비교 연산자
>= : 좌항이 우항보다 크거나 같은지 확인하는 대소 비교 연산자
<= : 좌항이 우항보다 작거나 같은지 확인하는 대소 비교 연산자
= : 우항의 값을 좌항에 할당하는 대입 연산자

조건 연산자라는 것도 있습니다. A ? B : C와 같은 형태로 표현됩니다. 그러면 A가 참이면 B가 결과로, 거짓이면 C가 결과로 반환됩니다.

나중에 배울 논리 연산자(&&와 ||)와, 비트 연산자(&, |, ^, ~, <<, >>)도 있습니다.

한가지 더 알아봐야 할 것은 연산자의 우선순위입니다. C++에서도 수학에서와 같이 우선순위가 존재합니다. 산술 연산자는 곱셈, 나눗셈, 덧셈, 뺄셈 순으로 우선순위를 갖습니다. 비교 연산자는 산술 연산자보다 낮은 우선순위를 갖습니다. 대입 연산자는 모든 연산자 중에서 가장 낮은 우선순위를 갖습니다. 따라서 연산자 우선순위에 유의하며 코드를 작성해야 합니다.

이상으로 C++의 연산자에 대한 간단한 설명을 마치겠습니다. 감사합니다.
