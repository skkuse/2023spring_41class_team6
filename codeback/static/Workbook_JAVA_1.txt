자바(Java) 언어는 다양한 데이터 타입을 지원합니다. 데이터 타입을 이해하는 것은 자바 프로그래밍을 익히기 위해 매우 중요합니다. 이번 학습지에서는 자바 데이터 타입의 종류와 각각의 특징에 대해 상세히 알아보도록 하겠습니다.

데이터 타입(Data Types)은 컴퓨터가 변수(Variable)를 인식하는 방식을 나타냅니다. 각 변수의 데이터 타입은 해당 변수가 저장하는 데이터의 종류를 정의합니다. 자바의 데이터 타입은 크게 기본 자료형(Primitive Data Types)과 참조 자료형(Reference Data Types)으로 구분됩니다.

기본 자료형(Primitive Data Types)은 int, long, float, double, boolean, char 등이 있습니다. 이들은 기본값이 존재하며, 간단한 데이터 값을 나타내는 데 사용됩니다.

- int: 32비트 부호 있는 정수를 표현합니다. 기본값은 0입니다. 예제: `int age = 20;`
- long: 64비트 부호 있는 정수를 표현합니다. 숫자 끝에 L을 붙일 수 있습니다. 기본값은 0L입니다. 예제: `long distance = 123456789L;`
- float: 32비트 부동 소수점(float-point) 수를 표현합니다. 숫자 끝에 F나 f를 붙일 수 있습니다. 기본값은 0.0f입니다. 예제: `float pi = 3.14f;`
- double: 64비트 부동 소수점 수를 표현합니다. 기본값은 0.0d입니다. 예제: `double height = 181.2;`
- boolean: 논리적인 값(참(true)과 거짓(false))을 표현합니다. 기본값은 false입니다. 예제: `boolean isMyFriend = true;`
- char: 16비트 유니코드 문자를 표현합니다. 작은 따옴표(')로 묶어서 사용합니다. 기본값은 '\u0000'입니다. 예제: `char gender = 'M';`

참조 자료형(Reference Data Types)은 배열, 클래스, 인터페이스 등으로 이루어집니다. 변수는 해당 데이터 타입의 객체를 참조하는 것이며, 객체는 new 연산자를 사용해 메모리에 할당됩니다. 이들은 별도의 기본값을 가지지 않습니다.

예제:

```JAVA
// String은 객체 타입이므로 참조 자료형에 속합니다.
String name = "홍길동";
System.out.println(name); // 결과: 홍길동
```

자바에서 데이터 타입을 사용하는 방법에 대해 알아봤습니다. 이론적인 부분만 익힌다면 자바 프로그래밍을 할 때 불편한 상황이 많아집니다. 따라서 이번 학습지에서는 데이터 타입과 관련된 실제적인 예제를 작성해 분석해 보도록 하겠습니다.
