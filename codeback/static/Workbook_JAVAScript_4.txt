자바스크립트에서 반복문은 특정한 작업을 반복하여 수행하기 위해 사용됩니다. 반복문은 조건이 참인 동안 지정된 블록 내 코드를 실행합니다. 자바스크립트에는 for, while, do-while 세 가지 종류의 반복문이 있습니다.

for문은 초기화 구문, 조건 구문, 증가 또는 감소 구문을 모두 포함합니다. 이 구문들은 괄호 안에 세미콜론으로 구분되어 있습니다. for문의 사용 예는 다음과 같습니다.

```
for (var i = 0; i < 5; i++) {
  console.log(i);
}
```

이 코드는 변수 i를 0으로 초기화합니다. 그리고 i가 5보다 작은 동안 반복하고 i를 1씩 증가시킵니다. 각 반복에서는 i의 현재 값을 콘솔에 출력합니다.

while문은 조건만 포함합니다. while문의 사용 예는 다음과 같습니다.

```
var i = 0;
while (i < 5) {
  console.log(i);
  i++;
}
```

이 코드는 변수 i를 0으로 초기화합니다. 그리고 i가 5보다 작은 동안 반복합니다. 각 반복에서는 i의 현재 값을 콘솔에 출력하고 i를 1씩 증가시킵니다.

do-while문은 while문과 비슷하지만 조건 검사가 뒤에 있습니다. 즉, 먼저 코드 블록을 실행한 다음 조건을 검사합니다. do-while문의 사용 예는 다음과 같습니다.

```
var i = 0;
do {
  console.log(i);
  i++;
} while (i < 5);
```

이 코드는 변수 i를 0으로 초기화합니다. 그리고 코드 블록을 한번 실행한 다음에 i가 5보다 작은 동안 반복합니다. 각 반복에서는 i의 현재 값을 콘솔에 출력하고 i를 1씩 증가시킵니다.

이처럼 자바스크립트에서 반복문은 특정한 작업을 반복하여 수행하는 데 매우 유용합니다. 이러한 반복문을 적절하게 활용한다면 작성하는 코드의 양을 줄이고 작업을 보다 효율적으로 수행할 수 있습니다.
