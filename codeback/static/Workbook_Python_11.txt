안녕하세요. Python 언어의 초급 단계에서 File Handling에 대한 설명을 작성하겠습니다.

Python에서 파일 처리란, 컴퓨터 디스크에 저장된 파일을 읽기 혹은 쓰기하는 작업을 말합니다. Python은 파일을 다양한 모드로 열고, 읽기, 쓰기, 추가하기 등의 작업이 가능합니다.

일반적으로 open() 내장 함수를 통해 파일을 오픈하여 처리합니다. open() 함수에서 반환된 파일 객체를 이용하여, 다양한 파일 처리 작업을 수행할 수 있습니다.

파일 모드는 열 형식에 맞게 작성됩니다. 예를 들어, "r"은 읽기 모드, "w"는 쓰기 모드이며, "a"는 덧붙이기 모드입니다. 모드를 명시하지 않으면, 기본값으로 "r" 모드가 사용됩니다.

파일을 열어서 작업을 수행한 후, 파일을 닫는 것은 매우 중요합니다. 이렇게 하지 않으면 파일이 손상될 수 있습니다. 따라서, 파일을 닫는 작업은 파일 작업 후에 항상 수행해야 합니다. close() 내장 함수를 사용하여 파일을 닫을 수 있습니다.

아래는 파일을 읽기 및 쓰기 위한 예시입니다.

```python
# 파일을 쓰기 모드로 엽니다.
file = open("example.txt", "w")

# 파일에 내용을 씁니다.
file.write("This is an example file.\n")
file.write("Python is a great language.\n")

# 파일을 닫습니다.
file.close()

# 파일을 읽기 모드로 엽니다.
file = open("example.txt", "r")

# 파일의 내용을 출력합니다.
print(file.read())

# 파일을 닫습니다.
file.close()
```

위 코드에서는, "example.txt" 파일을 쓰기 모드("w")로 열어, 파일에 내용을 씁니다. 그리고 파일 객체를 닫습니다. 그 다음, 같은 파일을 읽기 모드("r")로 열어, 파일의 내용을 출력합니다. 마지막으로, 파일을 닫습니다.

위 예시에서는 파일을 읽기 위해 read() 함수를 사용하였습니다. 이 외에도, 파일에서 한 줄씩 읽어오기 위해 readline() 함수를 사용하거나, 리스트 형태로 한 번에 모두 읽어오기 위해 readlines() 함수를 사용할 수 있습니다.

또한, 파일에 내용을 추가하기 위해서는, "a" 모드를 사용합니다.

```python
# 파일을 추가 모드로 엽니다.
file = open("example.txt", "a")

# 파일에 내용을 추가합니다.
file.write("This is an appended line.\n")

# 파일을 닫습니다.
file.close()
```

위 코드에서는, "example.txt" 파일을 추가 모드("a")로 열어, 파일에 내용을 추가합니다. 그리고 파일을 닫습니다.

이렇게 Python에서 파일을 처리하는 방법을 익히면, 데이터 처리에 매우 유용하게 활용할 수 있습니다. 하지만, 파일 처리를 할 때는 항상 파일을 닫는 것을 잊지 말아야 합니다.
