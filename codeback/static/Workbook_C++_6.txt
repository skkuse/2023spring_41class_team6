C++은 데이터를 저장하고 처리하는 컴퓨터 프로그래밍 언어 중 하나로, C 언어의 확장된 형태로 개발되었습니다. 이 언어를 사용하면 다양한 종류의 데이터형을 다룰 수 있습니다. 

우선, 데이터형이란 변수가 저장하는 값의 종류를 의미합니다. C++에서는 다양한 데이터형을 제공하며, 정수, 실수, 문자 등을 저장할 수 있습니다.

정수형 데이터형에는 int, short, long, long long 등이 있습니다. 이들은 메모리에서 사용하는 비트 수에 따라 크기가 다릅니다. 예를 들어, int형 변수는 대부분의 시스템에서 4바이트 크기를 가지며, -2,147,483,648 ~ 2,147,483,647 사이의 값을 저장할 수 있습니다.

실수형 데이터형에는 float, double, long double 등이 있습니다. float는 단정밀도를 갖는 실수형이며, 대부분의 시스템에서 4바이트 크기를 가지며 소수점 이하 6자리까지 정확도를 가집니다. double은 배정밀도를 갖는 실수형으로 float보다 크기는 두배지만 소수점 이하 15자리까지 정확도를 가집니다.

문자형 데이터형에는 char이 있습니다. char형 변수는 1바이트 크기를 가지며, 문자나 숫자 하나를 저장할 수 있습니다.

또한, bool형은 true 또는 false 값을 가지는 논리형 데이터형입니다.

이제, 데이터형을 사용하여 입출력을 해보겠습니다.

먼저, 사용자로부터 입력받는 경우 cin 함수를 사용합니다. 

```
int num;
cout << "숫자를 입력하세요: ";
cin >> num;
```

위 코드에서는 사용자로부터 입력받은 값을 num 변수에 저장하는 예시입니다. 사용자가 입력하는 값은 모두 문자열로 인식되기 때문에, int형 변수에 저장하기 위해서는 반드시 형 변환을 해주어야 합니다.

출력하는 경우에는 cout 함수를 사용합니다.

```
int num = 10;
cout << "입력한 숫자는 " << num << "입니다."; 
```

위 코드에서는 변수 num에 저장된 값을 출력하는 예시입니다. << 연산자를 사용하여 여러 값을 한 번에 출력할 수도 있습니다.

총 5개의 데이터형(int, float, double, char, bool)만 설명하였지만, C++에서는 더 다양한 데이터형을 제공합니다. 적절한 데이터형을 선택하여 프로그래밍을 해야 합니다.
