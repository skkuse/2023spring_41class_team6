C++ 언어에서 참조자(Reference)란 변수나 객체의 또 다른 이름으로서, 참조자를 통해 해당 변수나 객체에 직접적인 접근이 가능합니다. 즉, 참조자를 이용하면 포인터를 통해 해야했던 간접적인 접근을 바로 가능하게 만들어줍니다. 

C++로 작성된 다음 예시를 살펴보겠습니다.

``` c++
#include <iostream>

int main() {
    int num = 10;
    int &numRef = num;

    std::cout << "num = " << num << std::endl;    // num = 10
    std::cout << "numRef = " << numRef << std::endl;  // numRef = 10

    numRef = 20;

    std::cout << "num = " << num << std::endl;    // num = 20
    std::cout << "numRef = " << numRef << std::endl;  // numRef = 20

    return 0;
}
```

위 예시에서 int형 변수 num을 선언하고, int형 참조자 numRef를 num으로 초기화하였습니다. 이후에 출력문을 통해 num과 numRef의 값을 확인해보면 두 값이 모두 10으로 출력됩니다. 

그러나 numRef를 통해 값을 20으로 변경하였을 때, 위 출력문에서 num과 numRef의 값이 모두 20으로 변경된 것을 볼 수 있습니다. 이는 C++에서의 참조자가 해당 변수나 객체에 대한 별칭으로, 메모리 주소값을 참조하여 해당 값을 변경한 것입니다.

참조자는 포인터와 비슷한 개념으로, 상황에 따라 참조자를 사용해야하는 경우가 있습니다. 예를 들어, 함수 호출시 복사 생성자(Copy Constructor)가 호출될 때 객체의 크기가 크거나, 객체 생성에 시간이 오래 걸리는 경우 포인터보다 참조자를 사용하면 더 효율적인 코드를 작성할 수 있습니다. 

C++에서의 참조자는 const 참조자, 레퍼런스 매개변수, 참조자 반환(Char) 등 다양한 경우에 사용될 수 있습니다. 따라서 C++을 공부하면서 참조자를 숙지하고, 포인터와의 차이점을 이해하는 것이 중요합니다.
